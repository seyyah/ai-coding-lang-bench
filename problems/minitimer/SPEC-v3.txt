1-) OVERVIEW

mini-timer is a command-line time tracking tool.
It stores time records in a plain text file and supports
initializing the workspace, starting/stopping/resuming timers, 
viewing logs with duration calculations, and managing tracked tasks.

All data is persisted in a directory called .minitimer/
in the current working directory.

2-) COMMANDS

--- init ---
Usage: python minitimer.py init
Creates a .minitimer/ directory and an empty timers.dat file.
If .minitimer/ already exists, print "Already initialized" and exit.

--- start ---
Usage: python minitimer.py start "Study Physics"
Appends a new task to timers.dat with status RUNNING.
Each task has: id (auto-increment), description, status, start_time, end_time.
Prints "Started timer #<id> for: <description>"

--- stop ---
Usage: python minitimer.py stop 1
Changes timer #1 status from RUNNING to STOPPED and records the end time.
Prints "Timer #1 stopped."
If the timer does not exist, prints "Timer #1 not found."
If the timer is already STOPPED, prints "Timer #1 is already stopped."

--- resume ---
Usage: python minitimer.py resume 1
Re-activates a STOPPED task by changing its status back to RUNNING.
Sets a new start_time and clears the previous end_time.
Prints "Resumed timer #1."
If the timer is already RUNNING, prints "Timer #1 is already running."

--- log ---
Usage: python minitimer.py log
Prints all timers with status and duration information:
  [1] [RUNNING] Study Physics (Started: 14:00)
  [2] [STOPPED] Coding Project (10:00 - 12:30 | Duration: 2h 30m)
If no timers exist, prints "No timers found."

--- stats ---
Usage: python minitimer.py stats
Prints summary statistics and total time spent on all completed tasks.
Example output:
  Total tasks tracked: 5
  Completed tasks: 3
  Total duration spent: 6h 45m
If no timers exist, prints "No timers found."

--- clear ---
Usage: python minitimer.py clear 1
Removes timer #1 from timers.dat.
Prints "Cleared timer #1."

--- clear-all ---
Usage: python minitimer.py clear-all
Wipes all records from timers.dat after a user confirmation.
Prints "All timers have been cleared."

3-) DATA FORMAT

File: .minitimer/timers.dat
Format: One timer per line, fields separated by |
Example:
1|Study Physics|RUNNING|14:00|NONE
2|Coding Project|STOPPED|10:00|12:30

4-) ERROR HANDLING

Running any command before "init": print "Not initialized. Run: python minitimer.py init"
Invalid command: print "Unknown command: <command>"
Missing arguments: print "Usage: python minitimer.py <command> [args]"
Invalid ID: If a non-existent ID is provided for stop/resume/clear, print "Timer #<id> not found."
